Keypair

new Keypair(client)

Keypair API wrapper.

Parameters:
Name Type Description
client Client

the Client API wrapper object to bind

Methods

info(accessKey, fields)

Information of specific Keypair.

Parameters:
Name Type Description
accessKey string

Access key to query information. If client is not authorized as admin, this will be ignored and current API key infomation will be returned.
fields array

Fields to query. Queryable fields are: 'access_key', 'secret_key', 'is_active', 'is_admin', 'user_id', 'created_at', 'last_used', 'concurrency_limit', 'concurrency_used', 'rate_limit', 'num_queries', 'resource_policy'.

list(userId, fields)

List Keypairs of given user ID.

Parameters:
Name Type Default Description
userId string null

User ID to query API keys. If user ID is not given and client is authorized as admin, this will return every keypairs of the manager.
fields array

Fields to query. Queryable fields are: "access_key", 'is_active', 'is_admin', 'user_id', 'created_at', 'last_used', 'concurrency_used', 'rate_limit', 'num_queries', 'resource_policy'.

add(userId, isActive, isAdmin, resourcePolicy, rateLimit, accessKey, secretKey)

Add Keypair with given information.

Parameters:
Name Type Default Description
userId string null

User ID for new Keypair.
isActive boolean true

is_active state. Default is True.
isAdmin boolean false

is_admin state. Default is False.
resourcePolicy string default

resource policy name to assign. Default is default.
rateLimit integer 1000

API rate limit for 900 seconds. Prevents from DDoS attack.
accessKey string null

Manual access key (optional)
secretKey string null

Manual secret key. Only works if accessKey is present (optional)

mutate(accessKey, input)

mutate Keypair for given accessKey.

Parameters:
Name Type Description
accessKey string

access key to mutate.
input json

new information for mutation. JSON format should follow: { 'is_active': is_active, 'is_admin': is_admin, 'resource_policy': resource_policy, 'rate_limit': rate_limit }
